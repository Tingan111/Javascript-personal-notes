關於這個專案

這是一個基本的網頁結構示範，包含了最常見的四個主要區塊：導航列、側邊欄、主要內容區和頁尾。

每個區塊都有其特定的功能和樣式設計，這是現代網站的標準架構。

結構說明

Nav (導航列)：放置網站主要導航連結，通常固定在頁面頂部。

Sidebar (側邊欄)：放置分類選單、搜尋功能或相關連結。

Main Content (主要內容)：網站的核心內容區域。

Footer (頁尾)：放置版權資訊、聯絡方式等次要資訊。
